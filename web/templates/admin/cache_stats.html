Cache Information

{{if .Data.IsRedis}}

Redis Distributed Cache

Using Redis for distributed caching, enabling cache sharing across multiple application instances. Ideal for production deployments.

{{else}}

In-Memory Cache

Using local in-memory caching. Fast but not shared between application instances. Set OCMS_REDIS_URL to enable Redis.

{{end}}

Site Configuration Cache

Caches site-wide configuration values (site name, description, logo, etc.). Automatically invalidated when configuration is updated.

Sitemap Cache

Caches the generated sitemap.xml. Regenerated hourly or when content changes. Reduces database queries for large sites.

Menus & Languages Cache

Caches navigation menus and language configurations for fast frontend rendering. Automatically invalidated when updated.
